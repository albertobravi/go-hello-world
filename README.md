## Setup

```
git clone https://github.com/albertobravi/go-hello-world.git ~/projectDir
```

```
cd ~/projectDir
```

```
export GOPATH=$HOME/projectDir
```

```
go install github.com/albertobravi/hello
```

```
./bin/hello
```
